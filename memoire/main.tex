\chapter*{Résumé}
\addcontentsline{toc}{chapter}{Résumé}

Ce mémoire présente \textbf{Scientific Watch Agent}, un système multi-agents fondé
sur des grands modèles de langage (LLM) conçu pour automatiser la veille scientifique.
À partir d'un sujet de recherche, le système extrait, filtre, résume et synthétise
automatiquement la littérature académique pertinente.

La contribution centrale du mémoire est la démonstration de la \textbf{généralisation
cross-domaine} : le système s'adapte à de nouveaux domaines de recherche sans
ré-entraînement spécifique, grâce à une approche de méta-apprentissage qui prédit
les paramètres de configuration optimaux à partir de caractéristiques du domaine
(\textit{meta-features}).

Les principales contributions techniques sont :
\begin{itemize}
  \item Un pipeline multi-agents complet orchestré par LangGraph, intégrant sept
    agents spécialisés (QueryExpander, QualityCritic, Summarizer, Synthesizer, etc.) ;
  \item Un scorer de qualité scientifique à six dimensions (venue, auteurs, impact,
    vélocité, récence, pertinence) optimisé par AutoML (Optuna TPE) ;
  \item Les patterns Reflexion et ReAct appliqués à la synthèse et à l'expansion
    de requêtes ;
  \item Une évaluation méta-learning sur 19 domaines par validation croisée
    leave-one-out (LOO-CV), confirmant l'hypothèse centrale H3
    (\textit{Borda Count} : 15/19 = 79\,\%, $p = 0.014$, effet large $r_{rb} = +0.57$).
\end{itemize}

\textbf{Mots-clés :} veille scientifique, systèmes multi-agents, LLM, LangGraph,
méta-learning, AutoML, Optuna, Reflexion, ReAct, NDCG.

\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This dissertation presents \textbf{Scientific Watch Agent}, a multi-agent system
based on large language models (LLMs) designed to automate scientific literature
monitoring. Given a research topic, the system automatically retrieves, filters,
summarises, and synthesises relevant academic publications.

The central contribution is the demonstration of \textbf{cross-domain generalisation}:
the system adapts to new research domains without domain-specific retraining,
through a meta-learning approach that predicts optimal configuration parameters
from domain characteristics (meta-features).

Key technical contributions include a seven-agent LangGraph pipeline, a six-feature
quality scorer optimised by Optuna TPE, Reflexion and ReAct patterns, and a
meta-learning evaluation across 19 domains confirming hypothesis H3
(Borda Count: 15/19 = 79\,\%, $p = 0.014$, large effect $r_{rb} = +0.57$).

\textbf{Keywords:} scientific monitoring, multi-agent systems, LLM, LangGraph,
meta-learning, AutoML, Optuna, Reflexion, ReAct, NDCG.
